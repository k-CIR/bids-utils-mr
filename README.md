# cir-utils
Providing an easy to use interface for organizing your raw data on SPICE to BIDS.

1. Clone this repository to your project folder on SPICE.
2. Clone the repository: [cir-utils-serve](https://github.com/k-CIR/cir-utils-serve) to your local machine to connect to the service running on SPICE.

All processing is done on SPICE. This repository provide an interface by serving a html-file (over ssh) to your local machine that makes it easy to construct a config file and process different data modalities in your project.

See the cir-wiki pages: https://k-cir.github.io/cir-wiki/SPICE/05_spiceBIDS/ for a detailed description on how to use the interface.
